Quick Reference¶

Usage:

from pyio import SimulationData

data = SimulationData.from_list(range(20), path='test/',
                                header='INS_Rayleigh_Benard_hdf5_plt_cnt_')

data.fields[20.0 : 60.0 : 2]['temp', 'pres'][:, :, :, :, :]
